# 测试登录信息

## 快速体验Generate页面

由于Generate页面需要登录才能访问，你可以使用以下方式之一：

### 方法1：创建新账户
1. 访问 http://localhost:8080/generate
2. 点击 "Sign Up" 标签页
3. 填写任意邮箱和密码（6位以上）
4. 点击 "Sign Up" 创建账户

### 方法3：Google登录（如果已配置）
点击 "Continue with Google" 按钮
